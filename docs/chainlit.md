# 🔬 论文复现助手

## Paper Reproduction Agent

---

### 我能做什么

| 能力 | 说明 |
|------|------|
| 📄 **搜索论文** | 自动搜索 ArXiv、Semantic Scholar 等来源 |
| 📖 **阅读论文** | 提取论文全文，理解实验配置和方法 |
| 🔍 **定位源码** | 找到论文对应的官方开源实现 |
| 📥 **克隆仓库** | 自动克隆代码到本地工作区 |
| 💬 **论文问答** | 基于论文内容进行深度问答 |

---

### 使用方式

直接告诉我你想复现的论文名称，例如：

- *"复现 Attention Is All You Need"*
- *"帮我复现 ResNet 论文"*
- *"找一下 LLaMA 的官方实现"*

Agent 会自动完成搜索 → 阅读 → 定位源码 → 克隆的全流程。

---

### 执行过程

Agent 执行时，你会在对话中看到一个 **可折叠的执行面板 (Execution Panel)**，展示每一步的详细操作和终端输出。它不会干扰主对话 —— 你可以继续提问，或者在完成后查看细节。
